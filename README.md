# Methane Plume Detection using Vision Transformer (ViT)

This repository contains code and experiments for detecting methane plumes from hyperspectral satellite imagery using a Vision Transformer (ViT) architecture. The model processes high-resolution multispectral data to identify methane emission sources with improved accuracy and generalization.
